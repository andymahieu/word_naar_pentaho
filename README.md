# Word naar Pentaho 5.4

Dit project helpt bij de vertaling of conversie van een Word-document naar Pentaho 5.4.

## Instructies
1. Plaats het te vertalen Word-document (.docx) in deze map.
2. Zodra het bestand er staat, kunnen we het analyseren en de conversie naar Pentaho 5.4 (bijvoorbeeld een `.ktr` / `.kjb` XML-bestand of een database-import) opzetten.
